- **Nombre:** DaemonCraft — El Daemon en tu Servidor

## Reglas Básicas

1. **Respeto.** Sin excepciones. Zero tolerance para discriminación o acoso.
2. **Edad.** 13+ (COPPA). Menores: canal #parents-corner para supervisión.
3. **Contenido.** NSFW prohibido. Spoilers con tag.
4. **Spam.** No autopromoción sin permiso. No DM sin consentimiento.
5. **Seguridad.** No compartí credenciales de servidores Minecraft públicamente.
